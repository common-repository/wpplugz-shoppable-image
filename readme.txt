== Description ==

Introducing the Shoppable Images plugin for WooCommerce! Turn stunning product visuals into interactive experiences that drive sales.

**What it does:**

- Create beautiful shoppable images showcasing multiple products in a single look.
- Effortlessly tag products directly on your image.
- Link each product tag to its individual product page for seamless purchasing.
- Design engaging shoppable images for targeted product collections or promotions.
- Increase conversion rates by letting customers visualize complete outfits or product pairings.
- Simplify product discovery with a visually captivating shopping experience.

**Benefits:**

- **Boost Sales**: Shorten the path to purchase with one-click access to products.
- **Enhance Engagement**: Keep visitors captivated by interactive and visually appealing product displays.
- **Showcase Collections**: Effectively promote curated product sets and inspire buying decisions.
- **Increase Customer Satisfaction**: Provide a user-friendly and intuitive shopping journey.
- **Works Flawlessly with WooCommerce**: Seamless integration for a smooth workflow.

**Create Shoppable Lookbooks in Minutes:**

- Upload your product image.
- Click anywhere on image to add product tags.
- Customize the look and feel of your tags for a cohesive style.
- Select shoppable image block to add to any post type.
- Or embed the shoppable image **shortcode** anywhere on your WooCommerce website.
- Watch your sales soar!

== Installation ==

= Minimum Requirements =

* WooCommerce 8.0.0 or greater
* PHP 7.4 or greater is required (PHP 8.0 or greater is recommended)
* MySQL 5.6 or greater, OR MariaDB version 10.1 or greater, is required

= Automatic installation =

Automatic installation is the easiest option -- WordPress will handle the file transfer, and you won’t need to leave your web browser. To do an automatic install of Shoppable Images, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”
 
In the search field type "Shoppable Images” then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.

= Manual installation =

Manual installation method requires downloading the Shoppable Images plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

= Updating =

Automatic updates should work smoothly, but we still recommend you back up your site.
